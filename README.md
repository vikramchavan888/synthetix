## Run Locally

Clone the project

```bash
  git clone https://github.com/vikramchavan888/synthetix
```

Go to the project directory

```bash
  cd synthetix
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
